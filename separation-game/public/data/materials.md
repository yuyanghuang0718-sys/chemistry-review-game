# 物質分析遊戲題庫來源

本題庫依據使用者提供的高中化學講義「掃描全能王 2025-09-23 19.44.pdf」整理。原則是以講義中出現的物質、例題、類題、比較表和圖片說明為主，不追求大量題目。

## 收錄原則

- 同一物質只保留具有教學價值的狀態。
- 狀態會影響導電性時，拆成不同題卡，例如 NaCl(s)、NaCl(l)、NaCl(aq)、HCl(g)、HCl(aq)。
- 判定以講義教法為準，不加入教材未明確處理的進階例外。
- 遊戲作答不再單獨測「組成粒子」；此概念由物質分類與狀態解析帶出。
- 導電性只在「此狀態是否導電？」測驗，不再放入物理性質複選。
- JSON 題庫位置：`separation-game/public/data/materials.json`

- 若日後講義新增酸鹼、有機或配位化合物，可新增 `tags`，但不需要修改 React component。
- 若教材沒有明確教導特殊例外，題庫應避免加入該物質或只保留教材明確使用的判定。
